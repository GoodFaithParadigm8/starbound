# starbound
Install truffle.
`npm i -g truffle`

Install gananche.
`npm i -g ganache-cli`

Run ganache.
`ganache-cli`

Open another terminal.
In the second terminal:

Build the contracts.
`truffle build`

Deploy the contracts.
`truffle deploy`

Install Dapp dependencies.
`cd client && npm i`

Start the Dapp.
`npm run dev`










This is NextJs ethereum starter Dapp.
